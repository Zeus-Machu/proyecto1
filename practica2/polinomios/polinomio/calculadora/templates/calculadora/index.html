POLINOMIOS

Un polinomio,
es una expresión algebraica compuesta por variables y constantes, combinadas mediante operaciones de suma, resta y multiplicación.

DEFINICION Y FUNDAMENTOS:
Un polinomio puede tener uno o más términos,
Cada término tiene una variable elevada a una potencia entera no negativa y un coeficiente numérico.

EJEMPLO:
Suma: (3x + 2) + (5x + 4) = 8x + 6
Resta: (3x + 2) - (5x + 4) = -2x - 2
Multiplicación: (3x + 2) * (5x + 4) = 15x² + 22x + 8
División: (3x + 2) / (5x + 4) = (3x + 2)/(5x + 4)
